В ветке Develop будет вестись вся разработка проекта. В ветке Master будут находится только релизы (пре-релизы) проекта.
Если вы закончили вести свою часть, сообщите всем и либо сделайте git merge, либо сообщите это Владиславу Пустовиту.
Каждая часть будет разрабатываться в разных ветка:
	1) mutator
	2) VirusTotal API
	3) UI/UX
Это требуется для комфортной разработки и чистого репозитория. Пожалуйста, соблюдайте эти правила!
