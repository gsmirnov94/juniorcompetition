Краткое руководство по Git:
Основные команды:
	1) git clone https://gitlab.com/team-viruses-unior-/viruses.git  -  копирует репозиторий к вам на компьютер и подключает вас к удаленному репозиторию
	2) git checkout <имя_ветки>  -  переключате вас на ветку, которую вы указали
	3) git status  -  показывает какие файлы изменены/созданы. Если они горят красным, то они не добавлены для коммита, если зеленым, то добавлены
	4) git add <имя_файла>  -  добавляет файл для коммита
	5) git commit -m "<комментарий_к_коммиту>"  -  делает коммит
	6) git push origin <имя_ветки>  -  отправляет ветку в удаленный репозиторий
Некоторые моменты:
	1) В файле .gitignore прописаны все файлы/папки, которые не нужны в проекте
	2) В файле .gitrequirements прописываются все зависимости, которых среди стандартных библиотек/модулей.
	   Комнадой pip install -r .gitrequirements установятся все зависимости.
